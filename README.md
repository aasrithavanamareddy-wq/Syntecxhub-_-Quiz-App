# Syntecxhub-_-Quiz-App
The Quiz App is a web-based application designed to test users' knowledge through multiple-choice questions. The application displays one question at a time with different answer options, allows users to select answers, and calculates the final score based on their responses. It includes features like Next/Submit buttons, score tracking, restart option, and interactive user interface.
The project is developed using HTML, CSS, and JavaScript. HTML is used to create the structure of the application, CSS is used for styling and responsive design, and JavaScript manages the quiz logic, question navigation, answer validation, and score calculation.
